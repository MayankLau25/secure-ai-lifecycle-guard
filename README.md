# Secure-AI-Lifecycle-Guard (SAIL-G)

[![Compliance: ISO 42001 & NIST AI RMF](https://img.shields.io/badge/Compliance-ISO%2042001%20%26%20NIST-green.svg)](https://github.com/MayankLau25/secure-ai-lifecycle-guard)

## Overview
**Secure-AI-Lifecycle-Guard (SAIL-G)** is a comprehensive framework and CI/CD plugin designed to enforce security standards throughout the AI development lifecycle. 

It implements the "White Coding" philosophy by ensuring that AI-assisted code is rigorously audited and that machine learning models are free from backdoors, data poisoning, and licensing risks.

## Key Features
- **AI-Code Auditor:** Scans code generated by AI models (Copilot, GPT) for common vulnerabilities like SQLi, XSS, and hardcoded secrets.
- **Model Integrity Scans:** Detects potential backdoors or adversarial neurons in neural network weights.
- **Data Privacy Guard:** Monitors training data pipelines for PII to prevent unintended data leakage in the final model.
- **Compliance as Code:** Enforces security policies based on global frameworks such as NIST AI RMF and ISO 42001.

## Professional Context
An essential tool for managing the **Secure AI Lifecycle**, reflecting leadership in AI governance, compliance, and engineering standards.
